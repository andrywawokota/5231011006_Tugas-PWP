Tentang Saya

Halo! Saya Yohanes Mario Andry Saputra Wawokota, seorang Web Developer yang memiliki minat besar dalam dunia teknologi dan desain. Saya senang menciptakan solusi digital yang menarik, fungsional, dan mudah digunakan, mulai dari website sederhana hingga aplikasi web yang lebih kompleks.

Saya memiliki pengalaman mengembangkan berbagai proyek secara individu maupun dalam tim, dengan fokus pada kualitas, efisiensi, dan pengalaman pengguna yang optimal. Saya memiliki semangat belajar tinggi dan selalu terbuka untuk tantangan baru, khususnya dalam bidang frontend development.

Keahlian saya meliputi HTML, CSS, JavaScript, serta backend dasar menggunakan PHP dan MySQL. Saya juga familiar dengan tools desain seperti Figma, serta framework seperti Bootstrap. Selain itu, saya aktif menggunakan Git dalam mengelola versi proyek secara kolaboratif.

Di luar aktivitas coding, saya menikmati membaca buku teknologi, menulis blog, dan aktif mengikuti komunitas pengembang untuk terus memperluas wawasan dan jaringan. Saya percaya bahwa pembelajaran adalah proses seumur hidup.
